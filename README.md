# TemporalSync (时间同步) ⏳

`TemporalSync`（时间同步）是一个面向智能医学、科研工作与大模型应用开发者的个人集成工作台与知识集成流。项目融合了极高美学水准的 **Basic Apple Guy** 极简主义设计系统、实时 AI 资讯集成、自习室实验性应用、微信公众号美化工具以及轻量化 Markdown 博客撰写系统。

---

## 🎨 极简主义设计系统 & 视觉系统

本站全面致敬 **Basic Apple Guy** 的精致极简美学，注重流畅的视觉过渡与通透的层级设计：
- **Canvas 双色画布** — 默认暗色模式（Dark Mode）采用深邃的夜空紫色（`#120D26`）；亮色模式（Light Mode）切换至温润饱满的苹果纯白（`rgb(250, 250, 250)`）。可通过左下角悬浮控制器无缝切换。
- **无边界粘性玻璃导航栏** — 顶部导航条固定定位（`sticky top-0`），去除下边界阴影与分割线，纯靠 `backdrop-blur-md` 虚化底层滑过的内容，实现无缝透明融合。
- **iOS 级呼吸感模式切换** — 独创“临时过渡类”拦截算法，在主题切换的瞬时注入 `0.8秒` 的阻尼缓出过渡（`cubic-bezier(0.25, 1, 0.5, 1)`），使得明暗交替平滑如丝，而日常超链接 Hover 依然保持极速响应。
- **排版与字号规范** — 菜单、标题全部使用 `Roboto Condensed` 字体并强制转换为 uppercase（大写）；普通正文、描述采用 `myriad-pro` 字体，且**字号统一强制对齐为 `15.3px`**，保证完美的视觉舒适度。
- **扁平化资讯卡片** — AI 热点卡片去除全部边框和大量重阴影，以扁平底色形态嵌入时间轴中；左侧时间轴竖线重构为 4px 宽的半透明玻璃柱效果，具有极高折射感。

## ✨ 核心版块

### 1. 关于 / 个人主页 (`/`)
- 以 GSAP 驱动的流畅 SplitText 动画展现文字主旨。
- 采用微弱灰线格栅和响应式双栏，展现个人工作室思考日志。

### 2. AI 热点 (`/hot`)
- 自动聚合 LLM 开发进展、AI 医疗、科研动态等前沿资讯。
- 卡片完全扁平化且右侧边界长线消融，由玻璃时间轴进行轻盈排布。

### 3. 自习室 (`/work`)
- 用于展示独立应用和实验性原型的个人沙盒。
- 包含高度自适应的 `md2red` 小红书卡片生成器，支持 Markdown 实时解析预览。

### 4. 博客系统 (`/writing`)
- 升级为双列大图网格布局，卡片带平滑圆角，完全复刻 `basicappleguy.com` 列表样式。
- **定制占位封面**：当文章正文无图时，自动呈现 TSYNC 专属品牌占位图，白天为冷灰渐变，夜晚为深邃紫渐变，并在 hover 时带微幅缩放动效。
- **极简居中详情**：详情页重构为居中排版，上方为 tags · date 极轻元数据，中间为全大写粗标题，下方大图支持原始宽高显示。

### 5. 微信排版工具 (`/shiyun-wechat-md`)
- 彻底移除老旧单页 HTML，全面重构为子路由页面。
- 将 Markdown 一键渲染为契合“时韵自习室”风格的微信排版格式，支持一键复制代码及一键复制 Inline CSS 富文本。

### 6. 控制中心 / 设置 (`/settings`)
- 支持修改系统主题主色调（Accent Color）、字号比例与中英双语切换。

### 7. 管理员后台 (`/admin`)
- 包含私密授权的安全校验控制台，支持云端文章管理。

## 🛠️ 技术栈

- **框架**：React 19 (TypeScript) + React Router v7 (Lazy-load 动态分包)
- **构建工具**：Vite 6 + Esbuild (服务端打包)
- **样式**：Tailwind CSS v4 (通过 `@tailwindcss/vite` 深度融合)
- **动画**：Motion.react + GSAP (ScrollTrigger)
- **云开发与数据**：腾讯云开发 (CloudBase JS SDK) 双向实时对接

## 🚀 本地运行与配置指南

### 1. 复制配置环境变量
在根目录下配置 `.env` 文件，填写云服务和 AI 资讯源密钥：
```env
MIMO_API_KEY="您的密钥"
MIMO_BASE_URL="https://api.xiaomimimo.com/v1"
MIMO_MODEL="mimo-v2"
```

### 2. 运行与构建
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 编译打包项目
npm run build
```
编译产物输出至 `dist/` 文件夹下。
